<h1>go-lox - an implementation of the lox programming language from "crafting interpreters" in go.</h1>
<h2>building (on windows cmd): </h2>
<code>
  git clone https://github.com/collins994/go-lox.git
  cd go-lox 
  make build
  make run
</code>

<h2> running a file </h2>
<code>
	make run <filepath>
</code>
